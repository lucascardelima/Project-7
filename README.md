# Project 7 - GROUPOMANIA 

## Technologies Used in this project

### Front End

  - Vue.js
      - Vue Router
      - Pinia
      
  - Axios
  - Bootstrap 5
  - FontAwesome
    
### Back End 

  - Express
  - Node.js
  - JsonWebToken
  - Nodemon
  - MSSQL
  - Bcrypt
  - FS
  - Crypto
      - RandomUUID

### Data Base

  - Microsoft SQL Server
      - Stored Procedures

## Settings

### Front End

`cd groupomania/groupomania-frontend`

`npm run serve`

Port **8080**

### Back End

`cd groupomania/backend`

`nodemon server`

Port **3000**
